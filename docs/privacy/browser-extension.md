# BLREC 高光助手隐私说明

更新日期：2026 年 7 月 17 日

BLREC 高光助手用于连接用户自行部署的 BLREC 服务，在 B 站直播页面收录房间并标记高光。插件不运营开发者服务器，也不向 BLREC 项目维护者出售或传输用户数据。

## 处理的数据

- 用户填写的 BLREC 服务地址和管理员用户名。
- BLREC 返回的受限插件令牌。
- 当前 B 站直播房间号，以及用户点击操作时的直播标题、主播名称、播放器时间和操作时间。

插件不会读取或收集 B 站 Cookie、密码、弹幕内容、支付信息或其他网站的浏览记录。

## 使用与传输

房间号用于向用户自己的 BLREC 服务查询收录和录制状态。直播标题、主播名称和播放器时间仅用于用户主动执行收录或添加高光。上述数据只传输到用户填写的 BLREC 服务，不会与项目维护者、广告商或其他第三方共享，也不会用于广告、画像或分析。

公网 BLREC 地址必须使用 HTTPS。HTTP 仅允许 localhost、本地域名和私有局域网地址。

## 存储与删除

BLREC 地址、用户名和受限令牌保存在浏览器扩展的本地存储中，直到用户撤销授权、清除扩展数据或卸载插件。房间和高光记录保存在用户自己的 BLREC 数据库中，由用户通过 BLREC 管理或删除。

用户可在 BLREC 的“设置 → 浏览器插件授权”中撤销令牌。撤销后插件无法继续访问 BLREC，重新连接前不会恢复。

## 权限说明

- `storage`：保存 BLREC 地址、用户名和受限插件令牌。
- `https://live.bilibili.com/*`：在 B 站直播页面显示收录和高光按钮，并读取完成这些功能所需的直播信息。
- 可选的 HTTP/HTTPS 主机权限：连接用户明确填写并授权的 BLREC 服务地址。

## 联系方式

如有隐私问题，请在 [BLREC GitHub Issues](https://github.com/luweiCN/blrec/issues) 中联系项目维护者。提交 Issue 时请勿附带访问令牌、Cookie 或其他凭据。
