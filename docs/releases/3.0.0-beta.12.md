# BLREC 3.0.0-beta.12

这是面向群晖及其他 Docker 环境的公开测试版。自动化测试和生产构建已经通过，但尚未完成真实环境验收；本版继续在真实群晖环境验证录制中剪辑、投稿与弹幕回灌。

## 高光剪辑

- 剪辑页改为播放器与时间轴一体化工作台，每次只编辑一个分 P，时间轴紧贴视频并支持拖动播放位置。
- 时间轴按需生成少量真实画面预览，悬浮可查看大图；预览使用串行抽帧和有界缓存，避免持续占用 NAS 资源。
- 新增明确的“新建片段”入口、播放器内开始/结束按钮以及 `I`、`O` 快捷键。
- 选中待创建或已创建片段时自动回填名称和范围；按秒调整边界时播放器同步跳到对应画面。
- 修复录制中增长文件的无损裁剪定位，并允许失败片段原地重试。

## 弹幕与网络

- 隐私保护导致 UID 为 0 的普通弹幕不再被误判为系统消息。
- 单个分 P 的弹幕 XML 缺失时跳过该分 P，其他可用弹幕仍继续回灌。
- 普通响应读取超时不再错误判定整条网络线路失效；连接建立失败仍会触发线路容灾。

升级前请备份 `/cfg` 对应目录和部署 `.env`。
