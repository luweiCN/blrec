# BLREC 3.0.0-beta.23

这是以主列表布局和实时刷新体验为重点的公开测试版。自动化验证已完成，但尚未完成真实环境验收。

## 主列表布局

- 房间管理、录制任务、上传任务和片段管理取消 `1180px` 最大宽度，使用侧栏以外的完整可用空间。
- 四个主列表统一桌面端 `24px`、窄屏 `12px` 的内容留白，筛选区和列表边界保持一致。
- 修复片段管理内容超过视口后被外层容器截断、无法向下滚动的问题。

## 实时任务刷新

- 上传和录制任务的状态变化继续即时更新当前行；后台校准不会清空已有列表或反复显示加载状态。
- 新任务和当前页任务移除仍会自动同步；分页外任务退出五分钟实时快照时不再刷新当前页。
- 房间管理继续直接应用实时快照，片段管理仍只在用户操作成功后刷新一次，没有增加轮询频率。

## 升级说明

本版本不新增数据库迁移、环境变量或目录挂载，可从 `3.0.0-beta.22` 直接升级。升级前仍应备份 `/volume1/docker/blrec-next/config` 与当前 Compose 文件。
