# BLREC 3.0.0-beta.5

这是面向群晖及其他 Docker 环境的公开测试版。自动化测试和生产构建已经通过，但尚未完成真实环境验收。

## 新功能

- 新增 Arc/Chromium 本地浏览器插件，在 B 站直播页提供“收录”“收录并投稿”和“添加高光”。
- 新增仅能访问插件接口的受限令牌，并支持在 BLREC 设置页查看和撤销插件授权。
- 高光书签可映射到录像时间轴，无损裁剪视频和对应弹幕，并创建独立的投稿任务。
- GitHub Release 会附带浏览器插件 ZIP；插件采用开发者模式本地加载，不通过 Chrome Web Store 分发。

## 修复

- 修复浏览器 Service Worker 调用原生 `fetch` 时可能出现 `Illegal invocation` 的问题。
- 修复短房间号收录后未切换真实房间号，以及录制状态变化后按钮未及时更新的问题。

升级仅替换镜像，不需要迁移设置；升级前仍应备份 `/cfg` 对应的宿主目录和部署 `.env`。更新插件时无需重启浏览器，在扩展管理页重新加载后刷新 B 站直播页面即可。
