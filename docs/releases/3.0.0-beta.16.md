# BLREC 3.0.0-beta.16

这是面向群晖及其他 Docker 环境的公开测试版。自动化测试、静态检查和生产构建已经通过，但尚未完成真实环境验收；本版将继续进行长期运行观察。

## 上传稳定性

- 预上传会话从单个分 P 开始逐步放行，创建窗口最高为 5；遇到 HTTP 406/429 时自动收缩、冷却并探测恢复，不再暂停等待人工重试。
- 投稿响应因网络或容器重启无法确认时，持续按标题与远端文件名核对近期稿件，不会盲目重复提交；明确未发出的请求仍自动重试。
- 自动投稿使用 ffprobe 的真实媒体时长过滤不足 60 秒的异常录像碎片；文件保留在本地，BVC 21588 会尽量标出对应分 P。

## 弹幕与列表

- 弹幕 WebSocket 使用登录态优先、匿名兜底，以改善匿名消息流不完整的问题。
- 本地视频播放与分页弹幕列表合并，支持随播放高亮、自动滚动、按需加载和点击弹幕跳转。
- 房间、录制和上传列表使用复合信息单元，减少横向滚动，并显示投稿可见性、合集、弹幕采集与回灌进度。

## 永久片段库

- 侧边栏新增“片段管理”，可查看来源、时长、大小和投稿进度，并支持预览、下载、投稿、重试与删除。
- 片段使用独立 `/clips` 挂载，永久保留且不计入录像容量或自动清理；升级时必须同步更新 Compose 并创建宿主机 `clips` 目录。
