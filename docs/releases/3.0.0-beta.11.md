# BLREC 3.0.0-beta.11

这是面向群晖及其他 Docker 环境的公开测试版。自动化测试和生产构建已经通过，但尚未完成真实环境验收；本版将进入持续运行测试，请保留旧录像作为投稿失败时的人工兜底。

## 房间设置

- 房间设置弹窗统一为标准宽度，保留录制、弹幕和后处理选项。
- 网络请求和录像删除策略统一读取全局设置，不再提供容易产生冲突的房间级覆盖项。
- 标题关键词仍可按房间设置，直播标题命中任意关键词时才开始录制。

## 投稿与弹幕

- 弹幕回灌中断后自动重新排队，不再要求用户判断“视为已发送”或“接受重复风险”。
- 投稿任务页面移除仅供开发排查的稿件设置核验差异，详细结果继续写入日志。
- 新增 biliupforjava 房间规则迁移工具，可保留仅自己可见、标题/简介/分P模板、投稿分区、动态开关、弹幕回灌和录像删除策略。

## 部署迁移

- 新增旧 BLREC 配置合并工具的全量迁移支持，迁移前自动备份新版设置和 SQLite 数据库。
- 迁移工具不会复制 Cookie、账号 Token、API Key 或通知密钥；投稿账号继续使用新版中已扫码登录的主账号。
- Java 合集只有合集 ID、没有新版要求的分区 ID；远程自定义封面也不能直接对应本地素材库，这两项会明确报告并留待重新选择，不会静默写入错误配置。

升级前请备份 `/cfg` 对应目录和部署 `.env`。
