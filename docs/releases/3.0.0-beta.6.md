# BLREC 3.0.0-beta.6

这是面向群晖及其他 Docker 环境的公开测试版。自动化测试和生产构建已经通过，但尚未完成真实环境验收，发布后计划持续观察 3～5 天。

## 修复

- 修复弹幕令牌与 WebSocket 身份不一致时，已确认开播的房间仍无法开始录像的问题。
- 录像启动不再等待弹幕连接；匿名与账号 Cookie 方式均失败时，视频仍会继续录制。
- 弹幕连接每场直播优先使用完整匿名链路，失败后只尝试一次账号 Cookie；同场重连保持成功身份，下一场重新从匿名开始。
- 修复 NAS 上本地视频播放和高光剪辑接口已经完成、页面却一直显示加载中的问题。
- 浏览器插件现在会校验已保存的授权，连接有效时只显示地址、管理员用户名和“修改”按钮。
- 收紧网络管理表格宽度并移除面板的重复横向滚动条，桌面宽度下可直接看到操作列。

升级只替换镜像，不需要迁移设置。升级前仍应备份 `/cfg` 对应的宿主目录和部署 `.env`；本地浏览器插件需要在 Arc 扩展管理页点击“重新加载”。
