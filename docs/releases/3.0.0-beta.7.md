# BLREC 3.0.0-beta.7

这是面向群晖及其他 Docker 环境的公开测试版。自动化测试和生产构建已经通过，但尚未完成真实环境验收；升级后仍需持续观察录制、剪辑和投稿流程 3～5 天。

## 录制与投稿

- 房间管理、录制任务和上传任务现在相互独立；直播结束时才按最新的任务、房间设置决定是否创建上传任务。
- 录制任务可以单独覆盖投稿参数，高光片段会生成各自独立的上传任务。
- 录像文件索引可在异常重启后恢复，本地播放不再无限等待，并会自动打开首个可用分段。

## 高光剪辑

- 浏览器工具支持为高光点命名，并把正常直播缓冲作为基线，只校准用户主动回看的时间。
- 剪辑页支持在时间轴设置开始和结束位置、同时准备多个片段、预览范围及无损裁剪。
- 关键帧检查自动执行；只有需要明显向前多保留内容时才要求确认。
- 已创建片段可跨页面刷新恢复，并显示上传状态、进度和审核通过后的稿件入口。

本版本会自动执行数据库迁移。升级前请备份 `/cfg` 对应目录和部署 `.env`；本地浏览器插件需要在 Arc 扩展管理页重新加载新版 `dist` 或 Release 压缩包。
