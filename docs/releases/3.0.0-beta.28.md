# BLREC 3.0.0-beta.28

这是针对高光剪辑播放器全屏画面布局的公开测试版。自动化验证已完成，但尚未完成真实环境验收。

## 全屏画面布局

- 网页全屏和系统全屏共用严格分离的两行布局：上方只显示视频，下方完整容纳按钮行和时间轴。
- 视频脱离网格行高计算，只铺满上方实际可用区域；底部控制区增高或滚动时不会覆盖视频。
- 视频继续使用 `object-fit: contain` 居中等比缩放，在不裁切任何画面的前提下尽可能放大，剩余空间显示黑边。
- 全屏容器按实际视口边界伸展，并为设备底部安全区保留空间。

## 升级说明

本版本不新增数据库迁移、环境变量或目录挂载，可从 `3.0.0-beta.27` 直接升级。升级前仍应备份 `/volume1/docker/blrec-next/config` 与当前 Compose 文件。
