# BLREC 3.0.0-beta.18

这是针对 NAS 持续运行中暂停任务和旧片段记录的公开测试版修复。自动化验证已完成，但尚未完成真实环境验收。

## 上传恢复

- 媒体索引与上传快照串行化，上传必须等待文件索引稳定。
- 精准恢复已投稿但被旧版页码校验误暂停的任务，只继续审核检查，不重复上传。

## 片段清理

- 旧 `/rec/highlights` 记录的文件已经不存在时，可正常删除数据库记录。
- 仍保留路径边界检查，不会删除 `/clips` 之外现存的视频或弹幕文件。

本版本不新增数据库迁移、环境变量或目录挂载，可直接从 `3.0.0-beta.17` 升级。
