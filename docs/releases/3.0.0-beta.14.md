# BLREC 3.0.0-beta.14

这是面向群晖及其他 Docker 环境的公开测试版。自动化测试和生产构建已经通过，但尚未完成真实环境验收；本版用于验证分 P 增量预上传和高光时间轴修正。

## 分 P 增量预上传

- 直播中的分 P 完成并稳定后立即预上传，上传任务会同步出现在列表中。
- 直播结束后读取最新投稿设置，只补传尚未完成的分 P，并提交一个包含全部有效分 P 的稿件。
- 支持容器重启后续传、关闭或重新开启本场投稿、账号变化后安全重传，以及尾部分 P 处理失败后的自动恢复。
- 列表通过 SSE 实时显示任务增删、已预上传分 P 数量和传输进度。

## 高光时间轴

- 待编辑范围与主轨、已创建片段与独立轨道保持对齐，边界手柄跨越两条轨道。
- 操作面板和高光说明使用全局浮层，不再被播放器边界截断。
- 连续选择高光点或调整边界时，操作面板会跟随当前锚点；悬浮虚线在片段范围上也能稳定显示。

升级前请备份 `/cfg` 对应目录和部署 `.env`。
