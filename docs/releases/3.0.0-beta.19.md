# BLREC 3.0.0-beta.19

这是针对片段投稿交互和任务创建延迟的公开测试版修复。自动化验证已完成，但尚未完成真实环境验收。

## 片段投稿

- 片段稿件标题默认使用片段名称，同时允许在投稿前修改。
- 片段投稿不再展示分 P 标题模板和本地删除策略，生成的片段仍永久保存在片段目录中。
- 新建 B 站合集时可在当前弹窗直接上传封面，新封面上传完成后自动选中。

## 上传任务

- 创建片段上传任务不再停止并等待正在运行的大文件上传，设置确认后会直接进入上传队列。
- 片段上传任务显示最终投稿标题，本地片段管理仍保留原片段名称。

本版本不新增数据库迁移、环境变量或目录挂载，可直接从 `3.0.0-beta.18` 升级。
