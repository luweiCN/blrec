# BLREC 3.0.0-beta.17

这是 `3.0.0-beta.16` 的公开测试版兼容性补丁，修复自动排除短录像分片后，原始 P 序号与 B 站实际提交页码不一致造成的误判。自动化验证已经完成，但尚未完成真实环境验收。

## 分 P 核对

- 审核与转码修复按实际提交顺序核对远端页码，同时继续使用远端 filename 绑定 CID，避免稀疏分片被错误暂停。
- 投稿设置核验只比较真正提交的分片标题；缺少原始分片索引的旧任务快照也可按实际提交顺序完成转码修复。
- 录制任务中的线上分 P 链接按 B 站实际页码生成。

本补丁不新增数据库迁移、环境变量或目录挂载，可直接从 `3.0.0-beta.16` 升级。
