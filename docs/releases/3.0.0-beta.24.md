# BLREC 3.0.0-beta.24

这是以本地录像播放与高光剪辑体验为重点的公开测试版。自动化验证已完成，但尚未完成真实环境验收。

## 播放与全屏

- 高光剪辑新增只占满网页的网页全屏，同时保留浏览器的系统全屏。
- 系统全屏中顶部操作区仅在鼠标悬浮或键盘聚焦时显示，底部时间轴以浮层显示，不再截断视频画面。
- 高光剪辑和录像播放弹窗支持空格播放/暂停、方向键快进/快退及调节音量、`M` 静音；高光剪辑还支持 `F` 切换网页全屏。

## 状态记忆与故障恢复

- 播放器音量初始为 50%，用户调整后由当前浏览器统一记住。
- 每个分 P 的播放位置保存在本机，播放中每 5 秒更新一次，暂停、切换或关闭时会立即保存，播放完成后清除。
- 仅在浏览器明确报告不支持编码时显示编码错误；媒体源或网络的瞬时故障会自动重载一次并恢复原进度。
- “重新打开录像”现在会销毁当前媒体实例并重新打开原文件，同时保留播放位置。

## 升级说明

本版本不新增数据库迁移、环境变量或目录挂载，可从 `3.0.0-beta.23` 直接升级。升级前仍应备份 `/volume1/docker/blrec-next/config` 与当前 Compose 文件。
