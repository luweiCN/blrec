# BLREC 3.0.0-beta.13

这是面向群晖及其他 Docker 环境的公开测试版。自动化测试和生产构建已经通过，但尚未完成真实环境验收；本版用于继续验证录制中剪辑、投稿与弹幕回灌。

## 高光剪辑

- 播放器和自定义时间轴改为单一工作区，播放、定位、高光标记和片段编辑保持同步。
- 支持同时保留多个蓝色待编辑片段，点击空白处只取消选中，不会丢失草稿。
- 已创建片段以绿色显示，可复制为新草稿后继续微调。
- 起止点和高光点均在时间轴内操作，支持按秒调整、全屏编辑及录制中不稳定尾部保护。
- 移除耗时的时间轴缩略图生成，降低 NAS 上大文件剪辑的等待与资源占用。

## 录制与投稿

- 录制中的本场上传状态改为读取当前房间配置，避免“自动投稿已开启”与“本场不上传”显示冲突。
- 加强片段名称、异步创建、录制中安全边界及高光点关联的一致性处理。

升级前请备份 `/cfg` 对应目录和部署 `.env`。
