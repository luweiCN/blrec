# BLREC 3.0.0-beta.2

这是面向群晖多网络录制与投稿场景的公开测试版。

本版本已通过自动化测试和 Docker 冒烟，但尚未完成真实环境验收。请先备份配置与数据库，并仅用 3～5 个房间连续观察数天。

## 主要变化

- 自动发现可用网卡、源地址和网关，支持按房间状态、弹幕、录像、上传及其他 B 站请求分别选线。
- 固定线路支持故障切换；状态轮询等短请求支持按请求轮换，单场录像始终保持线路粘性。
- 页面通过一条 SSE 连接实时显示房间状态、容器网速和上传进度。
- 支持按网卡限制上传速度，并记录选线、故障切换、录像、上传、审核和投稿配置核验日志。
- 提供旧版设置安全迁移工具，不复制 Cookie、Webhook、通知密钥和废弃磁盘字段。

## 安装与升级

使用 Release 附带的 `compose.synology.yml` 和 `synology.env.example`。升级前同时备份 `/cfg`、SQLite 数据库和项目 `.env`，并按群晖部署文档执行固定版本升级或回滚。
