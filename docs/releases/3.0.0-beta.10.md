# BLREC 3.0.0-beta.10

这是面向群晖及其他 Docker 环境的公开测试版。自动化测试和生产构建已经通过，但尚未完成真实环境验收；升级后仍需持续观察录制、剪辑、投稿与弹幕回灌流程。

## 高光剪辑与浏览器插件

- 时间轴紧贴视频预览；选择范围改为底部细线，已添加片段继续使用独立颜色显示。
- 录制中提示移到预览标题，减少视频与时间轴之间的干扰信息。
- 大录制缓冲最多累积 1 MiB 即刷新到操作系统，刚开录的视频可以更快被播放和剪辑读取。
- 高光名称输入框声明忽略 1Password 等密码管理器的自动填充。更新本地插件后需在 Arc 扩展管理页点击“重新加载”。

## 投稿与弹幕回灌

- 自动获取并缓存 B 站 WBI 签名密钥，修复弹幕回灌在发送前失败的问题。
- 请求准备完成后才将单条弹幕写为“发送中”；本地签名或请求构造失败可以安全重试。
- 启动时一次性恢复旧进程留下的全部发送中记录，并记录 CID、进度、尝试次数和恢复原因；无法确认远端结果的记录仍不会自动重发。
- 投稿核验兼容 B 站为转载稿件简介自动添加来源链接，不再产生错误的不一致提示。

升级前请备份 `/cfg` 对应目录和部署 `.env`。
