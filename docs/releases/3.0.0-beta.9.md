# BLREC 3.0.0-beta.9

这是面向群晖及其他 Docker 环境的公开测试版。自动化测试和生产构建已经通过，但尚未完成真实环境验收；升级后需持续观察录制、剪辑和投稿流程 3～5 天。

## 录制筛选

- 全局录制设置和单个房间设置新增“录制标题关键词”。
- 可填写多个关键词，直播标题命中任意一个才开始录制；留空时录制所有直播。
- 标题起初不匹配时继续监控，直播中改为匹配标题后会自动开始录制，并记录审计日志。

## 高光剪辑与投稿

- 剪辑入口按录像分 P 展示，避免多分 P 场次选错文件；时间轴支持拖动、标记吸附和相邻分 P 边界切换。
- 可一次准备多个片段，并对成品进行预览、下载和独立投稿。
- 片段投稿复用完整投稿表单，支持合集、封面、分区、创作声明、可见性、动态和定时发布；确认设置后才创建上传任务并立即排队。
- 修复预览与实际裁剪选择不同文件、新版投稿快照丢失创作声明、片段删除策略不生效等问题。

本版本会自动执行数据库迁移。升级前请备份 `/cfg` 对应目录和部署 `.env`；本地浏览器插件本次无需更新。
