# BLREC 3.0.0-beta.25

这是针对本地录像跳转稳定性和全屏显示的公开测试版。自动化验证已完成，但尚未完成真实环境验收。

## 录像跳转稳定性

- 修复点击或拖动进度条后，录像尾部单个非法 FLV 视频标签被误判成“当前浏览器无法播放”并销毁播放器的问题。
- 仅容忍已确认可跳过的非法 `codec id 0` 标签；浏览器真正不支持录像编码时仍会显示明确错误。
- 播放器错误处理会等待 mpegts 当前事件队列结束后再执行，避免恢复或重开时出现二次播放器异常。

## 网页全屏

- 底部按钮区和时间轴作为一个完整区域参与全屏布局，不再覆盖视频画面。
- 视频在扣除完整底部区域后的可用空间内按原始比例最大化；允许黑边，但不会裁切画面。
- 顶部操作区仍会自动隐藏，鼠标移入或键盘聚焦后显示。

## 升级说明

本版本不新增数据库迁移、环境变量或目录挂载，可从 `3.0.0-beta.24` 直接升级。升级前仍应备份 `/volume1/docker/blrec-next/config` 与当前 Compose 文件。
