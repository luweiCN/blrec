# BLREC 录制与投稿

本上下文描述直播录制、录像管理与 B 站投稿之间的核心业务概念，用于避免把长期房间配置、单场录像和上传执行混称为“任务”。

## Language

**录制房间**：
长期存在的直播间配置，包含监控、录制和默认投稿规则。
_Avoid_: 任务、场次

**录制场次**：
主播一次连续开播对应的本地记录，聚合该次直播的所有分 P、弹幕和文件。
_Avoid_: 房间、上传任务

**投稿规则**：
录制房间为未来录制场次提供的默认投稿设置，不会自动追溯历史场次。
_Avoid_: 上传任务

**上传意图**：
某个录制场次是否计划投稿的明确决定，可继承投稿规则，也可由用户仅针对本场覆盖。
_Avoid_: 启用投稿规则

**上传任务**：
为一个计划投稿且文件就绪的录制场次创建的上传、投稿、审核及后续处理流程。
_Avoid_: 录制场次

**本地删除**：
终止某个录制场次的一切本地处理，并删除本系统中的记录与归属文件；它不删除或修改 B 站稿件。
_Avoid_: 删稿

**播放快照**：
打开录制中 FLV 时固定下来的文件大小、时长和关键帧索引；可在该范围内拖动，重新打开后才包含新增内容。
_Avoid_: 实时追播、边录边看
